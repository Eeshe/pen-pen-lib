# PenPenLib

PenPenLib is a Folia-compatible library that simplifies the development of Spigot/Paper Minecraft plugins.

## Features

- Full configuration of:
  - Messages
  - Sounds
  - Particles
  - Titles
- Powerful command and subcommand framework.

## Installation

1. Download the latest version of the PenPenLib JAR file from the releases page.
2. Drop the JAR file into the `plugins` folder of your Spigot/Paper server.
3. Restart the server to load the library.

## Compatibility

PenPenLib is compatible with Minecraft versions 1.19.4 and later. Compatibility with older versions is not currently planned, as Folia's oldest supported version is 1.19.4.

## Dependencies

- [PlaceholderAPI](https://github.com/PlaceholderAPI/PlaceholderAPI): For formatting placeholders within the `PlaceholderUtil` class.
